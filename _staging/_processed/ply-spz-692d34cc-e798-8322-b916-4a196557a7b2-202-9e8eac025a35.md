## Initial user need
我需要使用gsboxb把一个本地的ply转换为spz，windows系统。

## Latest user clarification

## Latest assistant answer
这个错误说明 **Windows 找不到 `gsbox` 的可执行文件**，所以命令无法运行。你需要做下面其中一种操作：


## ✅ 方法 1：在命令里使用 gsbox 的完整路径

如果你下载并解压后的 gsbox.exe 在这个目录，比如：

```
C:\tools\gsbox\gsbox.exe
```

那你运行命令时需要写成：

```powershell
"C:\tools\gsbox\gsbox.exe" p2z -i "D:\input.ply" -o "D:\output.spz" -ov 3
```

注意有空格的路径必须加引号 `" "`。


## ✅ 方法 2：把 gsbox.exe 所在目录加入系统 PATH

这样以后就能直接在任何地方输入 `gsbox` 来执行。
