# Unity 同时运行设置

## Initial user need
我使用unity开发了两个exe，现在同时在电脑打开时，只有选择到对应应用的窗口，它才会运行，怎么办？

## Latest user clarification
我就是使用Fullscreen Window，另一失焦的exe就会冻结。

## Latest assistant answer
你已明确使用的是 `Fullscreen Window`，但仍然出现**失焦时另一个 exe 冻结**的情况。说明有其他影响因素。我们可以**逐项排查**，确保符合 Unity 正常运行逻辑。


## ✅ 请你确认的关键点（务必全部检查）

### 1️⃣ **确保代码中设置了：`Application.runInBackground = true;`**
虽然你在 Player Settings 勾选了，但**强烈建议在代码中也加**，防止在某些平台下失效。

```csharp
void Start()
{
    Application.runInBackground = true;
}
```
